Dockerized Node Application

### Getting Started
1. `npm ci`
2. `npm test`
3. `npm start`

### Creating Docker Image/Container
1. `npm run docker:build`
2. `npm run docker:run`

1. Follow the [Setup](https://github.com/tgwittman/node-docker/blob/master/SETUP.md) documentation.
